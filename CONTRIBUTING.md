# Contributing to the Phytocure Research Dataset

Thank you for contributing to open cannabis science.

## Data Contribution Guidelines

### Requirements for New Records

1. **Cannabinoid profiles** — Must include a third-party ISO/IEC 17025 laboratory CoA
2. **Clinical references** — Must be published in a peer-reviewed journal with a valid DOI or PMID
3. **Dosage protocols** — Must be based on published clinical guidelines or RCT evidence
4. **Strain registry entries** — Must reference at least one verified cannabinoid profile

### How to Submit

1. Fork this repository
2. Add your data following the schema in [`methodology.md`](./methodology.md)
3. Include your ORCID ID in the PR description
4. Submit a pull request with a description of your data source

### Review Process

All submissions are reviewed by the Phytocure Research & Clinical Committee (RCC).
Review typically takes 14 days. Reviewers earn $PYCURE review bounties for approved contributions.

### Attribution

All contributors are listed in the CHANGELOG with their name and ORCID ID.
